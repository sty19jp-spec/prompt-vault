# prompt-vault

Public prompt vault for LLM workflows.

## Purpose
This repository stores prompt assets for reuse across ChatGPT, Gemini, and Claude.

## Structure
- OS: Global rules and constraints (prompt OS)
- Modules: Reusable components such as verification gates and output rules
- Templates: Task-specific prompt templates

## Policy
- ❌ Do NOT store secrets (API keys, tokens, private data).
- ✅ Text-based prompts only (Markdown).
- ✅ Public-safe content only.
